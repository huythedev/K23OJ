# K23OJ — ITK23 Nguyen Binh Khiem Online Judge

K23OJ is a customized fork of the [VNOJ](https://github.com/VNOI-Admin/OJ), which itself is built on top of the [DMOJ](https://github.com/DMOJ/online-judge) project.  
It provides a modern online judge platform for programming practice and competitive programming contests.

See it live at **https://k23oj.io.vn/**

---

## Features

K23OJ inherits the full functionality of DMOJ and VNOJ, including:

- Problem archive with rich metadata  
- Real-time judging with sandboxed environments  
- Contest hosting (ACM/ICPC, IOI, custom formats)  
- Submissions, rejudging, clarifications  
- Fully customizable templates (K23 theme modifications included)  
- Problem data stored locally (`DMOJ_PROBLEM_DATA_ROOT`)  
- Websocket-based live updates  
- Integrated judge → site cache sync  
- Vietnamese/English UI support  

For the base features, refer to the original list here:  
https://github.com/DMOJ/online-judge#features
